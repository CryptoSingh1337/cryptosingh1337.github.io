Project Detail:

It is basically a password storing application which stores your multiple passwords and provide you a single password to manage those stored passwords. It stores all of your passwords into the Oracle Database i.e. in Oracle 10g which you have to install on your system. This application can store multiple passwords one by one into Database. The main key is the USERNAME which must be unique for all user. On the basis of username it ADD, RETRIEVE, EXPORT, DELETE your passwords. This is just an initial commit to this project. Any changes/recommendations to this project are welcome. I hope you get the basic idea of making a password rack.

  • Export password file by taking the name of the file by user.
  • Password-Rack works on local database which is hosted on the Localhost of the system i.e. 127.0.0.1
  • Password-Rack ensures that user passwords are secured as it is on local server not on a remote server.
  • Password-Rack can store data more than 1000 records.
  • Very easy to use.
